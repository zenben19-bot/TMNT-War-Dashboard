Optional Member API Keys Setup

Faction member lists do not expose live energy, HP, medical cooldown, or drug cooldown. To show those columns, each member must give you their own API key with User: basic, bars, and cooldowns access. Paste one key per line below. Keys stay in this browser unless you press save.

Tip: Start with only a few keys to test. Minimal/custom keys should include User: basic, User: bars, and User: cooldowns. Torn rate limits API calls, so loading 20+ member keys can take a moment.

Enemy Threat / Suppression Setup

Optional manual intel improves scoring. Paste one enemy per line. Format: TornID, battle stats/spy estimate, note. Example: 1234567, 250m, active reviver. If no manual stats are entered, the engine scores using level, activity, status, and availability.
